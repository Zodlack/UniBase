Часть 1 (Теория)

Ответ: данный код, с задержкой в 3 секунды выводит четыре сообщения : 
Bad: 10
Bad: 12
Good: 15
Good: 21

Примечание: код выполнялся прямо в консоли браузера, то 
в этом контексте может не возникнуть ошибки.

В большинстве сред разработки и браузерах, высока вероятность того, 
что этот код вызовет ошибку "Cannot read property '4' of undefined"

Из-за асинхронной природы JavaScript, когда таймер срабатывает, значение переменной i может уже быть равно 4 (если массив имеет 4 элемента), 
что приведет к ошибке. Поэтому код внутри setTimeout будет пытаться обратиться к arr[4], 
что вызовет ошибку "Cannot read property '4' of undefined"

Модификация №1 

const arr = [10, 12, 15, 21];

for (let value of arr) {
  setTimeout(function () {
    console.log(value > 13 ? `Good: ${value}` : `Bad: ${value}`);
  }, 3000);
}


Модификация №2 

const arr = [10, 12, 15, 21];

arr.forEach((el) => setTimeout(() => {
    console.log(el > 13 ? `Good: ${el}`: `Bad: ${el}`);
},3000));
